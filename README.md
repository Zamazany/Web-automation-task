# Speeron E2E Automation Suite

This repository contains end-to-end automated UI tests for the Speeron Guest Portal using Playwright and NUnit (.NET 8.0).

## Features
- Menu Item validation (TV & Mobile Preview)
- Full menu navigation test
- Markdown documentation for test cases

## Setup & Run

1. Install .NET 8.0 SDK
2. Navigate to `/Speeron/` and run:
```bash
dotnet test
